# function-3
函数参数

1.形参：声明如下函数 
       
        function whatever(a,b,c){...}
        (1).如果实际传递参数数量大于函数声明的形参数量，超出的参数则不会配给形参名称;
        
            例：调用的时候执行whatever(1,2,3,4,5),参数1,2,3会分别赋值给a,b,c。而参数4和5则不会赋值给任何形参;
            
         (2).如果实际传递参数数量小于函数声明的形参数量，则没有对应参数的形参会赋值为underfined;   
            
            例：调用的时候执行whatever(1),参数1赋值给a,而b和c则会被赋值为undefined;
            
 2.隐式参数：所有的函数调用时都会传递两个隐式参数:arguments和this; 
  
        隐式参数不会显示列在函数签名里，但是它们默默地传递给函数并存在于函数作用域内。在函数内部，它们可以像其他显示命名的参数一样使用。
        
        arguments参数是传递给函索的所有实际参数的一个集合。
        
        例如
        
        function a(a,b,c,d){
        
        
        console.debug(arguments.length);     //2
        
        
        };
        
        
        a(1,2);
